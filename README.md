## Описание
Рекурсивный нисходящий парсер для вычисления битовых выражений с операторами: | (ИЛИ), & (И), << (сдвиг влево), >> (сдвиг вправо), ~ (НЕ) и скобками.

## Грамматика
```bnf
<expr>   ::= <or>
<or>     ::= <and> <or'>
<or'>    ::= "|" <and> <or'> | ε
<and>    ::= <shift> <and'>
<and'>   ::= "&" <shift> <and'> | ε
<shift>  ::= <unary> <shift'>
<shift'> ::= "<<" <unary> <shift'> | ">>" <unary> <shift'> | ε
<unary>  ::= "~" <unary> | <primary>
<primary>::= INT | "(" <expr> ")"
```

## Сборка
mkdir build

cd build

cmake ..

cmake --build .

cd ..

## Использование
.\build\Debug\expr_parser.exe

Введите выражение, например 1 << (2 & 3)

**ИЛИ**

.\build\Debug\expr_parser.exe input.txt

## Пример работы
Ввод: 1 << (2 & 3)

    Вывод: Result: 4
        Hex: 0x4


Ввод: ~0 & 255

    Вывод: Result: 255
        Hex: 0xff


Ввод: (1 | 4) >> 1

    Вывод: Result: 2
        Hex: 0x2

## Обработка ошибок
Программа обнаруживает и сообщает об ошибках:

Лексические: неизвестные символы, переполнение чисел

Синтаксические: незакрытые скобки, пропущенные операнды

Семантические: сдвиг на ≥32 бита
